# Cyber Threat Intelligence Lifecycle
### TLP
1. Red Light
   - Highly sensitive
   - Must be treated with high security
   - Must not be shared
2. Amber
   - Sensitive
   - Shared within own organization on a need to know basis
3. Green
   - Mildy sensitive
   - Can be shared with partners
   - Not shared publicly
4. White
   - Not sensitive
